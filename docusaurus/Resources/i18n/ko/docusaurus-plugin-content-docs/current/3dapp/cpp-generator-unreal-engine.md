### UMacros 생성기

언리얼 엔진에서 C++ 워크플로우를 향상시킵니다.

-   **변수, 배열 및 세트 생성**: 필수 변수(Bool, Int, Transform 등)를 생성하고 기본값을 쉽게 설정합니다.
-   **고급 플래그 및 지정자**: C++ 변수와 함수를 에디터에 노출시키면서 동작을 사용자 지정합니다.
-   **커스텀 노드 생성**: 블루프린트와 유사하게 다중 입력 및 출력이 있는 함수를 생성합니다.

### AActor 생성기

오류 없는 컴파일을 보장하면서 액터 계층 구조를 쉽게 생성하고 관리합니다.

-   **드래그 앤 드롭 관리**: 드래그 앤 드롭 기능으로 액터 계층 구조를 수정합니다.
-   **커스텀 컴포넌트 통합**: 생성된 액터에 커스텀 C++ 컴포넌트를 원활하게 통합합니다.
-   **전체 클래스 코드 생성**: 블루프린트를 생성하는 것만큼 빠르게 완전한 클래스 코드를 자동 생성합니다.

### C++ 도구상자

필수 개발 도구로 C++ 워크플로우를 간소화합니다.

-   **인클루드 및 델리게이트 찾기**: 엔진에서 include 및 delegate 시그니처를 검색합니다.
-   **즉각적인 로깅 및 디버깅**: 두 번의 클릭만으로 로그와 디버그 셰이프를 생성합니다.
-   **애셋 경로 가져오기**: 비 C++ 애셋을 코드에 쉽게 로드합니다.
-   **구조체 및 열거형 쉽게 생성**

### BP 노드 변환기

데이터베이스에서 널리 사용되는 노드를 검색하고 한 번의 클릭으로 동등한 C++ 노드를 얻습니다.
