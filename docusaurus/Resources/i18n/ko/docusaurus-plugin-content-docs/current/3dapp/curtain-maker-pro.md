# Curtain Maker Pro: 블렌더에서 실감나는 커튼 만들기

![Curtain Maker Pro for Blender](https://www.gfxcamp.com/wp-content/uploads/2025/09/Curtain-Maker-Pro.jpg)

:::tip[정보]
Curtain Maker Pro는 몇 초 만에 프로시저럴 커튼 메쉬를 생성하고 사용자 지정할 수 있도록 도와주는 무료 블렌더 애드온입니다. 인테리어 디자인, 건축 시각화, 또는 실감나는 드레이퍼리가 필요한 3D 장면에 완벽합니다.
:::

## 주요 기능

- **프로시저럴 생성**: 빠르게 실감나는 커튼 메쉬 생성
- **사용자 지정 가능한 매개변수**: 필요에 따라 다양한 속성 조정
- **인테리어 디자인 준비 완료**: 건축 시각화 프로젝트에 완벽
- **사용하기 쉬움**: 빠른 워크플로우를 위한 간단한 인터페이스

## 지원 블렌더 버전

광범위한 블렌더 버전 지원: 3.0 - 4.5
