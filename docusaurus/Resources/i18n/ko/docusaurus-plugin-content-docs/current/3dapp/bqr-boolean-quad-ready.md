# BQR Boolean Quad Ready: 블렌더용 사각 메쉬 최적화 플러그인

![Bqr Boolean Quad Ready](https://www.gfxcamp.com/wp-content/uploads/2025/05/Bqr-Boolean-Quad-Ready.jpg)

:::tip[정보]
BQR로 새로운 풀-쿼드 워크플로우를 빠르고 쉽게 활용하세요. 이 플러그인은 블렌더에서 불리언 모델링 중 생성되는 지저분한 와이어링을 최적화하고 줄여주며, 사각 와이어링을 최적화합니다.
:::

## 주요 기능

- **사각 메쉬 최적화**: 불리언 작업 중 사각 메쉬 위상을 최적화합니다.
- **와이어링 감소**: 불리언 모델링 중 생성되는 지저분한 와이어링을 줄입니다.
- **풀-쿼드 워크플로우**: 빠르고 쉬운 풀-쿼드 워크플로우를 가능하게 합니다.

## 지원되는 블렌더 버전

블렌더 버전 지원: 4.5, 4.4, 4.3.
