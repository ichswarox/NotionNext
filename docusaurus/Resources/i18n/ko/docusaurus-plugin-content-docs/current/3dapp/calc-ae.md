## 기능 목록

:::tip
Adobe After Effects를 떠나지 않고 빠르게 타임코드 계산을 할 수 있습니다. Calc AE는 정확한 결과를 보장하기 위해 컴퍼지션 프레임레이트를 자동으로 로드합니다. 수동으로 타임코드를 입력하거나 타임코드 보조 도구를 사용하여 일반적인 타임코드 값을 삽입할 수 있습니다.
:::
