## Cleanup Master v1.1.0의 주요 기능

- **셰입 레이어 최적화**: 애프터 이펙트에서 복잡한 셰입 레이어 관리를 간소화합니다
- **자동화된 클린업**: 지능적인 클린업 알고리즘으로 수동 작업을 줄입니다
- **워크플로우 효율성**: 모션 디자이너를 위한 반복 작업을 간소화합니다
- **크로스 버전 호환성**: 다양한 버전의 애프터 이펙트와 호환됩니다
- **시간 절약**: 셰입 레이어 유지 관리에 소요되는 시간을 크게 단축합니다

## 설치 가이드

1. jsxbin 파일을 다음 위치에 복사하세요: Support Files\Scripts\ScriptUI Panels
2. 애프터 이펙트에서 File → Scripts → Run Script File로 이동합니다
3. Cleanup Master 스크립트 파일을 선택합니다

## 문제 해결 팁

- **중국어 버전 AE 사용하지 마세요**: 중국어 버전은 표현식 오류를 일으킬 수 있습니다
- **스크립트 쓰기 권한 활성화**: 환경 설정에서 Edit → Preferences → General 패널로 이동하여 "Allow Scripts to Write Files and Access Network"를 체크합니다
- **버전 호환성**: 스크립트가 작동하지 않는다면 특정 AE 버전을 지원하지 않을 수 있습니다 (가능성 낮음)
