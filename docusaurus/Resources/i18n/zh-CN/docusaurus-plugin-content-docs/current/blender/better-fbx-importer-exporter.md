## 常见问题排除

### 安装问题
一些用户报告了安装错误。如果您遇到问题：
1. 解压ZIP文件
2. 打开解压的文件夹
3. 将内容重新打包到新的ZIP文件中
4. 安装新创建的ZIP文件

这个问题的发生是因为一些下载源添加了额外的文件夹层，而Blender要求插件使用单一文件夹结构。

### 纹理和材质问题
如果导出的FBX文件缺少纹理：
1. 选择文件 → 外部数据 → 将所有内容打包到.blend
2. 再次打开文件并选择解包所有内容
3. 或者，将所有路径设置为相对路径
4. 然后导出
