## 用于非真实感渲染的直接模型绘画系统

:::info
一个简单但强大的绘画系统，让您可以使用直观的画笔工具直接将材质应用到模型上，非常适合在Blender原生环境中创建非真实感(NPR)效果和风格化艺术作品。
:::

### 主要特点

- 使用画笔界面直接在3D模型上绘制材质
- 专为非真实感渲染(NPR)设计
- 与Blender原生工具集成的直观工作流程
- 用于创建风格化和艺术性渲染的灵活系统
- 为概念艺术家和插画师提供简化的绘画体验

### 系统要求

- **Blender**: 4.1或更高版本
- 兼容Windows、macOS和Linux

### 安装指南

1. 打开Blender首选项：编辑 → 首选项 → 插件
2. 点击右上角的下拉菜单
3. 选择"从磁盘安装"
4. 选择下载的ZIP文件
