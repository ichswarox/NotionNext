# Blender相机和光照设置插件 - Photographer 5.6.5 + 预设库

## 概述

Photographer插件使Blender中的灯光设置、相机对焦和各种参数调整变得更加简单。它支持EEVEE、Cycles和LuxCore渲染器，并包含Light Pack 01预设库。这个插件添加了新的物理相机和物理灯光设置，以及用于增强工作流程的LightMixer界面。

:::tip 主要特点
- 轻松调整相机对焦
- 灯光设置工具
- 参数调整控制
- 支持EEVEE、Cycles和LuxCore渲染器
- LightMixer界面
- 物理相机设置
- 物理灯光设置
- 包含Light Pack 01预设库
:::
