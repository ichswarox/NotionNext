# HairFlow: Blender毛髪アニメーションシミュレーションプラグイン v1.2.2

![Blender毛髪アニメーション用HairFlowプラグイン](https://www.gfxcamp.com/wp-content/uploads/2025/04/Hairflow.jpg)

:::tip[情報]
HairFlowは、Blenderの新しい毛髪システム用のアニメーションプラグインです。モディファイアーで生成された毛髪でも、ジオメトリノードを使用しない完成した毛髪でも、あらゆるカーブオブジェクトで動作します。キャラクターや動物に幅広く適用でき、毛髪シミュレーションを簡単で楽しくする多くの素晴らしい機能を備えています。
:::

## 対応Blenderバージョン

- Blender 4.5, 4.4, 4.3, 4.2, 4.1, 4.0
