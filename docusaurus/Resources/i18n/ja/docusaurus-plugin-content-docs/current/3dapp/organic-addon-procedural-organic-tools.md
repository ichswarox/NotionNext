Organic Addonは、Blenderでオーガニック形状を生成するためのプロシージャルで自動的なシステムです。あらゆるシーンに適応する動的な構造体を作成しましょう！

:::tip[主な機能]
- オーガニック形状を生成するためのプロシージャルで自動的なシステム

- Blender 4.0 - 4.5に対応
:::
