VFXアーティストのクリントン・ジョーンズ（[@\_Pwnisher\_](https://twitter.com/_Pwnisher_)）、ユニークなクリエイティブチャレンジを主催することで知られていますが、**「Rampage Rally」**コンテストのトップ100モンタージュをリリースしました。

このチャレンジは、アーティストにジャンプするビークルの提供されたテンプレートに基づいて独自の3Dシーンを作成することを招待しました。

<iframe width="100%" height="315" src="https://www.youtube.com/embed/yW5WIyEe_eE" title="YouTubeビデオプレーヤー" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

:::note クリエイターから
> 先月、2,625人の3DアーティストがRampage Rally 3Dアートチャレンジに参加し、そこでは私たち全員が同じ素朴なテンプレートから制作しました。30日後、トップ100のレンダリングが選ばれ、一緒に編集され、このモンタージュのためにスコアリングされました。お楽しみください！
:::
