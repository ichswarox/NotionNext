Zoo Tools Pro 2は、Maya用の強力なツールキットで、モデリング、ライティング、マテリアル、シェーダー、レンダリングのワークフローを強化する40以上のツールを提供します。

### 主な機能
*   **拡張機能:** Mayaに幅広い新機能を追加します。
*   **互換性:** Zoo Toolsがインストールされていない他のユーザーでも保存したファイルを開くことができます。
*   **バージョン2リライト:** 50以上の新しいツールを搭載した完全なリライト版。
*   **ユーザーフレンドリー:** 各ツールには独自のトレーニングページと包括的なツールチップが含まれています。
*   **簡単なインストール:** 簡単なドラッグアンドドロップインストーラー。

### インストール手順

<Tabs>
  <TabItem value="plugin" label="プラグイン">
    `zooInstall_2.11.1b`フォルダから`dragdropinstall.py`をMayaビューポートにドラッグしてインストールします。
  </TabItem>
  <TabItem value="presets" label="プリセット">
    プリセットフォルダを`C:\Users\yourUsername\Documents\zoo_preferences\assets\`にコピーします。
  </TabItem>
  <TabItem value="scenes" label="シーンファイル">
    1. Zooプラグインを開き、`Zoo Preferences > General > Zoo Preferences Folder`に移動してプリファレンスフォルダを開きます。
    2. `maya_scenes`、`maya_shaders`、`studioLibrary`フォルダを`zoo_preferences\assets\`ディレクトリにコピーします。
  </TabItem>
</Tabs>
