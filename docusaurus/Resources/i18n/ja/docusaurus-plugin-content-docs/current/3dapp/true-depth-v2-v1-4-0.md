## True Depth V2+V1.4.0 Blenderプラグイン - AI画像から3Dモデルへ

![True Depth Banner](https://www.gfxcamp.com/wp-content/uploads/2025/04/True-Depth.jpg)

### 概要

True Depthは、Blender内で任意の画像を直接3Dモデルに変換するAI搭載のプラグインです。高度なAIモデルを使用して、任意の画像やビデオファイルから信じられないほど詳細なデプスマップを生成し、それを使用して正確かつ自動的に3Dジオメトリに変換します。たったのワンクリックで！

### 特徴

:::info
- AI搭載の画像から3Dへの変換
- 詳細なデプスマップを生成
- ワンクリック3Dジオメトリ作成
- 画像とビデオをサポート
- 高度なAIモデル
:::

### システム要件

- Blender 4.5, 4.4, 4.3, 4.2, 4.1, 4.0, 3.6, 3.5, 3.4, 3.3, 3.2, 3.1, 3.0
