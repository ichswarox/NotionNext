### 概要

Infinite Retouchは、様々なレタッチング機能、ノイズ追加、シャープ化などを含むポートレートレタッチング用のPhotoshopプラグインです。

これは、あなたのワークフローに適合し、あなたのレベルに関係なく簡単にカスタマイズできるレタッチングパネルです。ワークフローレイヤーの作成からエクスポートまで、そしてその間のすべてのステップで、プロセスの各部分のための高度なツールを使用してください。

### 機能

:::info
- カスタマイズ可能なレタッチングパネル
- 複数のレタッチング機能
- ノイズ追加機能
- シャープ化ツール
- ワークフロー最適化
:::

### システム要件

- Photoshop CC 2021以降のバージョン
