## Gaffer - BlenderライトおよびHDRIコントロールプラグイン V3.2.5

Gafferは、シーンの照明を設定するのに役立つBlenderアドオンで、すべての適切な設定を便利なパネルに表示して、クイックアクセスと実験を可能にします。

### 機能

- パネルパラメーターを通じてシーンの照明とHDRIをより便利にコントロール・調整
- クイックアクセスと実験のための便利なパネル
- シーンの照明を効果的に設定するために設計

### 対応Blenderバージョン

- Blender 4.5, 4.4, 4.3, 4.2, 4.1, 4.0, 3.6, 3.5, 3.4, 3.3, 3.2, 3.1, 3.0, 2.93, 2.92, 2.91, 2.9, 2.83, 2.82, 2.81, 2.8, 2.79, 2.78

### インストール手順

<Tabs>
<TabItem value="blender4" label="Blender 4.x またはそれ以下" default>
1. 上部メニューで、編集 > プリファレンス > アドオン > インストールに移動
2. ポップアップウィンドウで、対応するZIPファイルを選択してインストール
</TabItem>
<TabItem value="blender41" label="Blender 4.1 またはそれ以上">
1. 上部メニューで、編集 > プリファレンス > アドオンに移動
2. 右上隅で「ディスクからインストール」を選択
3. ポップアップウィンドウで、対応するZIPファイルを選択してインストール
</TabItem>
