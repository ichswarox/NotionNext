## Motion Tools Pro 2.0.13 AEスクリプト

Motion Tools ProはMotion Design Schoolが開発したAEスクリプトで、AEキーフレームカーブの調整、センターポイントアライメント、そして弾性アニメーションエフェクトの生成が可能です。Motion Tools 2やMotion V2と類似したスクリプトです。

### 特徴

:::info[主な機能]
- **カーブエディタ**: よりスムーズなカーブ調整のためのエクスプレッションなしのカーブエディタ
- **アンカーポイント**: 1クリックでアンカーポイントを調整
- **スナップ**: オブジェクトを互いにスナップ
- **弾性**: 弾性アニメーションを生成
- **複製**: 複数のオブジェクトを複製し、位置を変更
- **スケール**: 複数のオブジェクトのスケールを変更
- **回転**: 複数のオブジェクトを回転
- **フェード**: 複数のオブジェクトをフェードイン/フェードアウト
- **カバー**: オブジェクトを2D/3Dカバー
- **スキュー**: オブジェクトをスキュー
- **モーション**: オブジェクトをモーション
- **スピード**: アニメーションのスピードを変更
- **タイミング**: キーフレームタイミングを変更
- **サイズ**: オブジェクトのサイズを変更
- **位置**: オブジェクトの位置を変更
- **回転**: オブジェクトの回転を変更
- **オーダー**: オブジェクトのオーダーを変更
- **ランダム**: オブジェクトをランダム化
- **カラー**: オブジェクトのカラーを変更
:::

### インストール手順

<Tabs>
<TabItem value="ae-script" label="AE Script">
1. インストール手順については、圧縮パッケージ内のドキュメントを参照してください
2. 中国語版AEの場合：スクリプトがエクスプレッションでエラーを報告する可能性があるため、中国語版は使用しないでください
3. 環境設定で、編集—環境設定—一般パネルから「スクリプトがファイルを書き込み、ネットワークにアクセスすることを許可」にチェックを入れてください
4. スクリプトがお使いのAEバージョンをサポートしていることを確認してください
</TabItem>
</Tabs>
