Chaos Vantageは、最も複雑な3Dシーンを探索するためのリアルタイムレイトレーシングプレビューソフトウェアです。Chaos® Vantageを使用すれば、もう待つ必要はありません。完全なV-RayシーンをVantageにドラッグするだけで探索を開始できます。また、3ds Max、SketchUp、Rhinoとのライブリンク機能により、リアルタイムビューポート使用も可能です。

## 主な機能

:::tip[主な機能]
- **設定不要**: V-RayシーンをChaos Vantageにドラッグするだけで閲覧を開始できます。以前のリアルタイムレンダリングソリューションとは異なり、ジオメトリの最適化、UVマッピング、ライトベイクは必要ありません。
- **大規模シーン**: Chaos Vantageは、数百万のポリゴンを持つ巨大なシーンを処理できます。複雑なV-Rayシーンをリアルタイムで表示する最速の方法です。
- **簡単なナビゲーション**: シーンを簡単に閲覧できます。3Dモデリングソフトウェアのコントロールモードやゲームのようなナビゲーションに似ています。自動衝突検出により、階段を上ったり、壁を通り抜けるのを避けたりできます。
- **100%レイトレーシング品質**: 完全なリアルタイムレイトレーシングレンダリング。Chaos Vantageは、物理ベースのカメラ、ライト、マテリアル、グローバルイルミネーションを使用して、最もリアルな結果を実現します。
- **迅速で簡単なアニメーション作成**: プレレンダリングに最適です。Chaos Vantageで直接アニメーションを作成、編集、レンダリングできます。アニメーションエディタを使用して、カメラアニメーション、トランジション、シーケンス順序の調整を簡単に作成できます。
- **ライブリンク**: 3ds Max、SketchUp、Rhinoとのライブリンクを通じて、シーンのリアルタイムレイトレーシングビューを取得し、アニメーションをレンダリングできます。シーンに加えた編集がChaos Vantageに直接表示されます。
:::
