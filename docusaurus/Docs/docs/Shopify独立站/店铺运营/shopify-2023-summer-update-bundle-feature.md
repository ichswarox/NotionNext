## Bundle 功能

因为这个功能目前已经发布，其他的还没上线，所以重点讲解组合套装功能。

-   [官方免费 App 链接](https://apps.shopify.com/shopify-bundles)
-   跟大部分人想的 Bundle 不一样，不是 Cross Sell，可以理解为一个新的商品
-   图片需要手动上传添加，不会自动带套装内商品的产品图片过去
-   会自动跟踪每款商品的库存（如果关闭了 "缺货后继续销售" 的话）
-   选项数量仍然不能超过 3 项
-   似乎有 bug
