## 固定移动端加购按钮

```css
<style>
    @media screen and (max-width: 450px){
        #id{
        position: fixed;
        bottom: 0px;
        left: 0;
        z-index: 99;
        height: 可选;
        }
    }
</style>
```

复制

## 如何实现点击加购按钮之后隐藏

```html
onclick="this.style.display = 'none'"
```

复制

## 如何隐藏 Buy it now 按钮

```css
.shopify-payment-button {
  display: none;
}
```

复制
